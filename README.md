# Private Diary

This is a personal diary storage service.  
It is deployed on heroku and can be accessed through the following address.  
https://django-private-diary.herokuapp.com/


## Install

```bash
$ pip install pipenv
$ pipenv install
```

## Usage

```bash
$ pipenv shell
```
